#SQL Server 2017 on Linux

##写在前面的话
>昨天老师说今天晚上上课要做实验使用SQL Server，可以选择带U盘和带电脑，而懒惰程序员总想什么都不带，于是考虑在Linux服务器搭建一个微软数据库。

##准备工作
>CentOS 7.3 以上服务器
>
>内存：1G以上
>
>硬盘：3G以上
>
>文件格式：XFS或者EXT4
